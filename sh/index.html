    Shell Commands with sh

    make is an orchestrator â€” it doesn't do much work itself, it calls other commands which do the work. That's what we recommend doing with jeeves as well. Among numerous ways to call other programs from Python we recommend a library named, quite concisely, sh: amoffat/sh.

    Installation

    It will be installed as an optional dependency of jeeves-shell[all].

    Alternatives

    See shell cobminators to check out other available tools.

    By default, it is bundled as a dependency for jeeves, and here is a simple example.

    jeeves.py
    import typer
from sh import grep, pip


def list_flake8_plugins():
    """List installed plugins for Flake8."""
    typer.echo(
        grep( # (1)!
            'flake8-',
            _in=pip.freeze(),
        ),
    )
    1. Equivalent to: pip freeze | grep flake8-
